# 🎬 Movie App

Movie search app built with React, TypeScript, Tailwind CSS, React Query and Zustand. Powered by TMDB API.

## Tech Stack

**Frontend:** React + TypeScript + Tailwind CSS + React Query + Zustand  
**Backend:** NestJS + PostgreSQL + Prisma + JWT + Swagger

## Features

- 🔐 Register / Login with JWT
- 🔍 Movie Search & Filter
- ❤️ Favorites
- ⭐ Reviews & Ratings
